Nombres, chapitre 26

Nb 26:1- Yahvé parla à Moïse et à Éléazar, fils d'Aaron, le prêtre. Il dit :
Nb 26:2- " Faites le recensement de toute la communauté des Israélites, par familles : tous ceux qui ont vingt ans et au-dessus, aptes à faire campagne en Israël. "
Nb 26:3- Moïse et Éléazar le prêtre les recensèrent donc, dans les Steppes de Moab, près du Jourdain vers Jéricho.
Nb 26:4- Comme Yahvé l'a ordonné à Moïse et aux Israélites à leur sortie du pays d'Égypte. Hommes de vingt ans et au-dessus :
Nb 26:5- Ruben, premier-né d'Israël. Les fils de Ruben : pour Hénok, le clan Hénokite ; pour Pallu, le clan Palluite ;
Nb 26:6- pour Hèçrôn, le clan Hèçronite ; pour Karmi, le clan Karmite.
Nb 26:7- Tels étaient les clans Rubénites. Ils comprenaient quarante-trois mille sept cent trente recensés.
Nb 26:8- Les fils de Pallu : Éliab.
Nb 26:9- Les fils d'Éliab : Nemuel, Datân et Abiram. Ce sont Datân et Abiram, hommes considérés dans la communauté, qui se soulevèrent contre Moïse et Aaron ; ils étaient de la bande de Coré quand elle se souleva contre Yahvé.
Nb 26:10- La terre ouvrit sa bouche et les engloutit ainsi que Coré, lorsque périt cette bande , lorsque le feu consuma les deux cent cinquante hommes. Ils furent un signe.
Nb 26:11- Les fils de Coré ne périrent pas.
Nb 26:12- Les fils de Siméon, par clans : pour Nemuel, le clan Nemuélite ; pour Yamîn, le clan Yaminite ; pour Yakîn, le clan Yakinite ;
Nb 26:13- pour Zérah, le clan Zarhite ; pour Shaûl, le clan Shaûlite.
Nb 26:14- Tels étaient les clans Siméonites. Ils comprenaient vingt-deux mille deux cents recensés.
Nb 26:15- Les fils de Gad, par clans : pour Çephôn, le clan Çephonite ; pour Haggi, le clan Haggite ; pour Shuni, le clan Shunite ;
Nb 26:16- pour Ozni, le clan Oznite ; pour Éri, le clan Érite ;
Nb 26:17- pour Arod, le clan Arodite ; pour Aréli, le clan Arélite.
Nb 26:18- Tels étaient les clans des fils de Gad. Ils comprenaient quarante mille cinq cents recensés.
Nb 26:19- Les fils de Juda : Er et Onân. Er et Onân moururent au pays de Canaan.
Nb 26:20- Les fils de Juda devinrent des clans : pour Shéla, le clan Shélanite ; pour Pérèç, le clan Parçite ; pour Zérah, le clan Zarhite.
Nb 26:21- Les fils de Pérèç furent : pour Hèçrôn, le clan Hèçronite ; pour Hamul, le clan Hamulite.
Nb 26:22- Tels étaient les clans de Juda. Ils comprenaient soixante-seize mille cinq cents recensés.
Nb 26:23- Les fils d'Issachar, par clans : pour Tola, le clan Tolaïte ; pour Puvva, le clan Puvvite ;
Nb 26:24- pour Yashub, le clan Yashubite ; pour Shimrôn, le clan Shimronite.
Nb 26:25- Tels étaient les clans d'Issachar. Ils comprenaient soixante-quatre mille trois cents recensés.
Nb 26:26- Les fils de Zabulon, par clans : pour Séred, le clan Sardite ; pour Élôn, le clan Élonite ; pour Yahléel, le clan Yahléélite.
Nb 26:27- Tels étaient les clans de Zabulon. Ils comprenaient soixante mille cinq cents recensés.
Nb 26:28- Les fils de Joseph, par clans : Manassé et Éphraïm.
Nb 26:29- Les fils de Manassé : pour Makir, le clan Makirite ; et Makir engendra Galaad : pour Galaad, le clan Galaadite.
Nb 26:30- Voici les fils de Galaad ; pour Iézer, le clan Iézrite ; pour Héleq, le clan Helqite ;
Nb 26:31- Asriel, le clan Asriélite ; Shékem, le clan Shékémite ;
Nb 26:32- Shemida, le clan Shemidaïte ; Hépher, le clan Héphrite.
Nb 26:33- Çelophehad, fils de Hépher, n'eut pas de fils, mais des filles ; voici les noms des filles de Çelophehad ; Mahla, Noa, Hogla, Milka et Tirça.
Nb 26:34- Tels étaient les clans de Manassé. Ils comprenaient cinquante-deux mille sept cents recensés.
Nb 26:35- Et voici les fils d'Éphraïm, par clans : pour Shutélah, le clan Shutalhite ; pour Béker, le clan Bakrite ; pour Tahân, le clan Tahanite.
Nb 26:36- Voici les fils de Shutélah : pour Érân, le clan Éranite.
Nb 26:37- Tels étaient les clans des fils d'Éphraïm. Ils comprenaient trente-deux mille cinq cents recensés. Tels étaient les fils de Joseph, par clans.
Nb 26:38- Les fils de Benjamin, par clans : pour Béla, le clan Baléite ; pour Ashbel, le clan Ashbélite ; pour Ahiram, le clan Ahiramite ;
Nb 26:39- pour Shephupham, le clan Shephuphamite ; pour Hupham, le clan Huphamite.
Nb 26:40- Béla eut pour fils Ard et Naamân : pour Ard, le clan Ardite ; pour Naamân, le clan Naamite.
Nb 26:41- Tels étaient les fils de Benjamin, par clans. Ils comprenaient quarante-cinq mille six cents recensés.
Nb 26:42- Voici les fils de Dan, par clans : pour Shuham, le clan Shuhamite. Tels étaient les clans de Dan, par clans.
Nb 26:43- Tous les clans Shuhamites comprenaient soixante-quatre mille quatre cents recensés.
Nb 26:44- Les fils d'Asher, par clans : pour Yimna, le clan Yimnite ; pour Yishvi, le clan Yishvite ; pour Béria, le clan Bériite.
Nb 26:45- Pour les fils de Béria : pour Héber, le clan Hébrite ; pour Malkiel, le clan Malkiélite.
Nb 26:46- La fille d'Asher se nommait Sarah.
Nb 26:47- Tels étaient les clans des fils d'Asher. Ils comprenaient cinquante-trois mille quatre cents recensés.
Nb 26:48- Les fils de Nephtali, par clans : pour Yahçéel, le clan Yahçéélite ; pour Guni, le clan Gunite ;
Nb 26:49- pour Yéçer, le clan Yiçrite ; pour Shillem, le clan Shillémite.
Nb 26:50- Tels étaient les clans de Nephtali, répartis par clans. Les fils de Nephtali comprenaient quarante-cinq mille quatre cents recensés.
Nb 26:51- Les Israélites étaient donc six cent un mille sept cent trente recensés.
Nb 26:52- Yahvé parla à Moïse et dit :
Nb 26:53- " C'est à ceux-ci que le pays sera distribué en héritage, suivant le nombre des inscrits.
Nb 26:54- A celui qui a un grand nombre, tu donneras un grand domaine, à celui qui a un petit nombre, un petit domaine ; à chacun son héritage, en proportion du nombre de ses recensés.
Nb 26:55- Toutefois, c'est le sort qui fera le partage du pays. Selon le nombre des noms dans les tribus patriarcales, on recevra son héritage ;
Nb 26:56- l'héritage de chaque tribu sera réparti par le sort en tenant compte du grand ou du petit nombre.
Nb 26:57- Voici, par clans, les Lévites recensés : pour Gershôn, le clan Gershonite ; pour Qehat, le clan Qehatite ; pour Merari, le clan Merarite.
Nb 26:58- Voici les clans de Lévi : le clan Libnite, le clan Hébronite, le clan Mahlite, le clan Mushite, le clan Coréite. Qehat engendra Amram.
Nb 26:59- La femme d'Amram se nommait Yokébed, fille de Lévi, qui lui était née en Égypte. Elle donna à Amram Aaron, Moïse et Miryam leur sœur.
Nb 26:60- Aaron engendra Nadab et Abihu, Éléazar et Itamar.
Nb 26:61- Nadab et Abihu moururent lorsqu'ils portèrent devant Yahvé un feu irrégulier.
Nb 26:62- Il y eut en tout vingt-trois mille mâles recensés, d'un mois et au-dessus. Car ils n'avaient pas été recensés avec les Israélites, n'ayant pas reçu d'héritage au milieu d'eux.
Nb 26:63- Tels furent les hommes que recensèrent Moïse et Éléazar le prêtre, qui firent ce recensement des Israélites dans les Steppes de Moab, près du Jourdain vers Jéricho.
Nb 26:64- Aucun d'eux n'était de ceux que Moïse et Aaron le prêtre avaient recensés, en dénombrant les Israélites dans le désert du Sinaï ;
Nb 26:65- car Yahvé le leur avait dit : ceux-ci mourraient dans le désert et il n'en resterait aucun, à l'exception de Caleb, fils de Yephunné, et de Josué, fils de Nûn.
