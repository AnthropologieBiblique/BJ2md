Proverbes, chapitre 10

Pr 10:1- Proverbes de Salomon. Le fils sage réjouit son père, le fils sot chagrine sa mère.
Pr 10:2- Trésors mal acquis ne profitent pas, mais la justice délivre de la mort.
Pr 10:3- Yahvé ne laisse pas le juste affamé, mais il réprime la convoitise des méchants.
Pr 10:4- Main nonchalante appauvrit, la main des diligents enrichit.
Pr 10:5- Amasser en été est d'un homme avisé, dormir à la moisson est d'un homme indigne.
Pr 10:6- Bénédictions sur la tête du juste, mais la bouche des impies recouvre la violence.
Pr 10:7- La mémoire du juste est en bénédiction, le nom des méchants tombe en pourriture.
Pr 10:8- L'homme au cœur sage accepte les ordres, l'homme aux lèvres folles court à sa perte.
Pr 10:9- Qui va honnêtement va en sécurité, qui suit une voie tortueuse est démasqué.
Pr 10:10- Qui cligne de l'œil donne du tourment, qui réprimande en face procure l'apaisement.
Pr 10:11- Source de vie : la bouche du juste, mais la bouche des impies recouvre la violence.
Pr 10:12- La haine allume des querelles, l'amour couvre toutes les offenses.
Pr 10:13- Sur les lèvres de l'homme intelligent se trouve la sagesse, sur le dos de l'insensé, le bâton.
Pr 10:14- Les sages thésaurisent le savoir, mais la bouche du fou est un danger menaçant.
Pr 10:15- La fortune du riche, voilà sa place forte; le mal des faibles, c'est leur indigence.
Pr 10:16- Le salaire du juste procure la vie, le revenu du méchant, le péché.
Pr 10:17- Il marche vers la vie, celui qui garde la discipline; qui délaisse la réprimande se fourvoie.
Pr 10:18- Les lèvres du menteur couvrent la haine; qui profère une calomnie est un sot.
Pr 10:19- Abondance de paroles ne va pas sans offense; qui retient ses lèvres est avisé.
Pr 10:20- La langue du juste est pur argent, le cœur des méchants est de peu de prix.
Pr 10:21- Les lèvres du juste repaissent une multitude, mais les fous meurent faute de sens.
Pr 10:22- C'est la bénédiction de Yahvé qui enrichit, sans que l'effort y ajoute rien.
Pr 10:23- C'est un jeu pour le sot de s'adonner au crime, et pour l'homme intelligent de cultiver la sagesse.
Pr 10:24- Ce que redoute le méchant lui échoit, ce que souhaite le juste lui est donné.
Pr 10:25- Quand la tourmente a passé, plus de méchant! mais à jamais, le juste est établi.
Pr 10:26- Vinaigre aux dents, fumée aux yeux, tel est le paresseux pour qui l'envoie.
Pr 10:27- La crainte de Yahvé prolonge les jours, les années du méchant seront abrégées.
Pr 10:28- L'espoir des justes est joie, l'espérance des méchants périra.
Pr 10:29- La voie de Yahvé est un rempart pour l'homme honnête, pour les malfaisants, une ruine.
Pr 10:30- Jamais le juste ne sera ébranlé, mais les méchants n'habiteront pas le pays.
Pr 10:31- La bouche du juste exprime la sagesse, la langue perverse sera coupée.
Pr 10:32- Les lèvres du juste connaissent la bienveillance, la bouche des méchants la perversité.
