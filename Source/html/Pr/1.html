Proverbes, chapitre 1

Pr 1:1- Proverbes de Salomon, fils de David, roi d'Israël :
Pr 1:2- pour connaître sagesse et discipline, pour pénétrer les discours profonds,
Pr 1:3- pour acquérir une discipline avisée - justice, équité, droiture -
Pr 1:4- pour procurer aux simples le savoir-faire, au jeune homme le savoir et la réflexion,
Pr 1:5- Que le sage écoute, il augmentera son acquis, et l'homme entendu acquerra l'art de diriger.
Pr 1:6- pour pénétrer proverbes et sentences obscures, les dits des sages et leurs énigmes.
Pr 1:7- La crainte de Yahvé, principe de savoir : les fous dédaignent sagesse et discipline.
Pr 1:8- Écoute, mon fils, l'instruction de ton père, ne méprise pas l'enseignement de ta mère :
Pr 1:9- c'est une couronne de grâce pour ta tête, des colliers pour ton cou.
Pr 1:10- Mon fils, si des pécheurs veulent te séduire, n'y va pas!
Pr 1:11- S'ils disent : " Viens avec nous, embusquons-nous pour répandre le sang, sans raison, prenons l'affût contre l'innocent;
Pr 1:12- comme le shéol, avalons-les tout vifs, tout entiers, tels ceux qui descendent dans la fosse!
Pr 1:13- Nous trouverons mainte chose précieuse, nous emplirons de butin nos maisons;
Pr 1:14- avec nous tu tireras ta part au sort, nous ferons tous bourse commune! "
Pr 1:15- Mon fils, ne les suis pas dans leur voie, éloigne tes pas de leur sentier,
Pr 1:16- car leurs pieds courent au mal ils ont hâte de répandre le sang;
Pr 1:17- car c'est en vain qu'on étend le filet sous les yeux de tout volatile.
Pr 1:18- C'est pour répandre leur propre sang qu'ils s'embusquent, contre eux-mêmes, ils sont à l'affût!
Pr 1:19- Tels sont les sentiers de tout homme avide de rapine : elle ôte la vie à ceux qu'elle habite.
Pr 1:20- La Sagesse crie par les rues, sur les places elle élève la voix;
Pr 1:21- à l'angle des carrefours, elle appelle, près des portes, dans la ville, elle prononce son discours :
Pr 1:22- " Jusques à quand, ô niais, aimerez-vous la niaiserie ? et les railleurs se plairont-ils à la raillerie ? et les sots haïront-ils le savoir ?
Pr 1:23- Convertissez-vous à mon exhortation, pour vous je vais épancher mon cœur et vous faire connaître mes paroles.
Pr 1:24- Puisque j'ai appelé et que vous avez refusé, puisque j'ai étendu la main sans que nul y prenne garde,
Pr 1:25- puisque vous avez négligé tous mes conseils et que vous n'avez pas voulu de mon exhortation,
Pr 1:26- à mon tour, je me rirai de votre détresse, je me moquerai quand viendra sur vous l'épouvante,
Pr 1:27- quand l'épouvante viendra sur vous comme l'orage, quand votre détresse arrivera comme un tourbillon, quand l'épreuve et l'angoisse fondront sur vous.
Pr 1:28- Alors ils m'appelleront, mais je ne répondrai pas; ils me chercheront et ne me trouveront pas.
Pr 1:29- Ils ont détesté le savoir, ils n'ont pas choisi la crainte de Yahvé,
Pr 1:30- ils n'ont pas voulu de mon conseil, ils ont méprisé toutes mes exhortations :
Pr 1:31- ils mangeront donc du fruit de leurs errements, ils se rassasieront de leurs propres conseils!
Pr 1:32- Car l'égarement des niais les tue, l'insouciance des sots les mène à leur perte;
Pr 1:33- mais qui m'écoute demeure en sécurité, il sera tranquille, sans craindre le malheur. "
