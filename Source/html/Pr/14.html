Proverbes, chapitre 14

Pr 14:1- La Sagesse bâtit sa maison, de sa main, la Folie la renverse.
Pr 14:2- Qui marche dans sa droiture craint Yahvé, qui dévie de ses chemins le méprise.
Pr 14:3- Dans la bouche du fou il y a un surgeon d'orgueil, les lèvres des sages les gardent.
Pr 14:4- Point de bœufs, mangeoire vide; taureau vigoureux, revenus abondants.
Pr 14:5- Le témoin véridique ne ment pas, mais le faux témoin exhale le mensonge.
Pr 14:6- Le railleur poursuit la sagesse, mais en vain, à l'homme intelligent le savoir est chose aisée.
Pr 14:7- Écarte-toi du sot, tu ignorerais les lèvres savantes.
Pr 14:8- Pour l'homme avisé, la sagesse est de surveiller sa conduite, mais la folie des sots n'est que tromperie.
Pr 14:9- Les fous raillent le sacrifice pour le péché, mais parmi les hommes droits se trouve la faveur.
Pr 14:10- Le cœur connaît son propre chagrin et nul étranger ne partage sa joie.
Pr 14:11- La maison des méchants sera détruite, la tente des hommes droits prospérera.
Pr 14:12- Tel chemin paraît droit à quelqu'un, mais en fin de compte c'est le chemin de la mort.
Pr 14:13- Dans le rire même, le cœur trouve la peine, et la joie s'achève en chagrin.
Pr 14:14- Le cœur dévoyé se rassasie de ses démarches, et l'homme de bien de ses œuvres.
Pr 14:15- Le niais croit tout ce qu'on dit, l'homme avisé surveille ses pas.
Pr 14:16- Le sage craint le mal et se détourne, le sot est insolent et sûr de lui.
Pr 14:17- L'homme prompt à la colère fait des sottises, l'homme malintentionné est odieux.
Pr 14:18- La part des niais, c'est la folie, les gens avisés se font du savoir une couronne.
Pr 14:19- Devant les bons, les méchants se prosternent, et aux portes des justes, les impies.
Pr 14:20- Même à son voisin, le pauvre est odieux, mais nombreux sont ceux qui aiment le riche.
Pr 14:21- Il pèche, celui qui méprise son prochain; heureux qui a pitié des pauvres.
Pr 14:22- N'est-ce pas s'égarer que machiner le mal ? Miséricorde et fidélité pour qui s'applique au bien.
Pr 14:23- Tout labeur donne du profit, le bavardage ne produit que disette.
Pr 14:24- Couronne des sages : leur richesse; la folie des sots est folie.
Pr 14:25- Un témoin véridique sauve des vies, qui profère des mensonges est un imposteur.
Pr 14:26- Dans la crainte de Yahvé, puissante sécurité; pour ses enfants il est un refuge.
Pr 14:27- La crainte de Yahvé est source de vie pour éviter les pièges de la mort.
Pr 14:28- Peuple nombreux, gloire du roi; baisse de population, ruine du prince.
Pr 14:29- L'homme lent à la colère est plein d'intelligence, qui a l'humeur prompte exalte la folie.
Pr 14:30- Vie du corps : un cœur paisible; mais l'envie est carie des os.
Pr 14:31- Opprimer le faible, c'est outrager son Créateur; c'est l'honorer que d'être bon pour les malheureux.
Pr 14:32- Par sa propre malice le méchant est terrassé, le juste trouve un refuge dans son intégrité.
Pr 14:33- En un cœur intelligent demeure la sagesse; on ne la reconnaît pas au cœur des sots.
Pr 14:34- La justice grandit une nation, le péché est la honte des peuples.
Pr 14:35- La faveur du roi va au serviteur intelligent et sa colère à celui qui fait honte.
