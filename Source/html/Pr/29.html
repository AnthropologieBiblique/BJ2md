Proverbes, chapitre 29

Pr 29:1- Celui qui, sous les reproches, raidit la nuque sera brisé soudain et sans remède.
Pr 29:2- Quand les justes se multiplient, le peuple est en liesse; quand les méchants dominent, le peuple gémit.
Pr 29:3- Qui aime la sagesse réjouit son père, qui hante les prostituées dissipe son bien.
Pr 29:4- Par l'équité, un roi fait prospérer le pays, mais l'exacteur le mène à la ruine.
Pr 29:5- L'homme qui flatte son prochain tend un filet sous ses pas.
Pr 29:6- Dans l'offense du méchant il y a un piège, mais le juste exulte et se réjouit.
Pr 29:7- Le juste connaît la cause des faibles, le méchant n'a pas l'intelligence de la connaître.
Pr 29:8- Les railleurs mettent la cité en effervescence, mais les sages apaisent la colère.
Pr 29:9- Un sage est-il en procès avec un sot, qu'il se fâche ou plaisante, il n'aura pas de repos.
Pr 29:10- Les hommes sanguinaires haïssent l'homme honnête, mais les hommes droits recherchent sa personne.
Pr 29:11- Le sot donne livre cours à tous ses emportements, mais le sage, en les réprimant, les calme.
Pr 29:12- Quand un chef accueille des rapports mensongers, tous ses serviteurs sont mauvais.
Pr 29:13- Le pauvre et l'oppresseur se rencontrent : tous deux reçoivent de Yahvé la lumière.
Pr 29:14- Le roi qui juge les faibles avec équité voit son trône affermi pour toujours.
Pr 29:15- Baguette et réprimande procurent la sagesse, le jeune homme laissé à lui-même est la honte de sa mère.
Pr 29:16- Quand se multiplient les méchants, le forfait se multiplie, mais les justes seront témoins de leur chute.
Pr 29:17- Corrige ton fils, il te laissera en repos et fera les délices de ton âme.
Pr 29:18- Faute de vision, le peuple vit sans frein; heureux qui observe la loi.
Pr 29:19- On ne corrige pas un esclave avec des mots : même s'il comprend, il n'obéit pas.
Pr 29:20- Tu vois un homme prompt au discours ? il y a plus à espérer d'un sot.
Pr 29:21- Si dès l'enfance on gâte son esclave, il deviendra finalement ingrat.
Pr 29:22- L'homme coléreux engage la querelle, l'homme emporté multiplie les offenses.
Pr 29:23- L'orgueil de l'homme l'humiliera, qui est humble d'esprit obtiendra de l'honneur.
Pr 29:24- C'est partager avec le voleur et se haïr soi-même, que d'entendre l'adjuration sans dénoncer.
Pr 29:25- Trembler devant les hommes est un piège, qui se confie en Yahvé est en sûreté.
Pr 29:26- Beaucoup recherchent la faveur du chef, mais de Yahvé vient le droit de chacun.
Pr 29:27- Abomination pour les justes : l'homme inique; abomination pour le méchant : celui dont la voie est droite.
