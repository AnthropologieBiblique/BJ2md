Proverbes, chapitre 21

Pr 21:1- Comme l'eau courante, le cœur du roi est aux mains de Yahvé qui l'incline partout à son gré.
Pr 21:2- Toutes les voies de l'homme sont droites à ses yeux, mais Yahvé pèse les cœurs.
Pr 21:3- Pratiquer la justice et le droit vaut, pour Yahvé, mieux que le sacrifice.
Pr 21:4- Regards altiers, cœur dilaté, flambeau des méchants, ce n'est que péché.
Pr 21:5- Les projets de l'homme diligent ne sont que profit; pour qui se presse, rien que la disette!
Pr 21:6- Amasser des trésors par une langue menteuse : vanité fugitive de qui cherche la mort.
Pr 21:7- La violence des méchants les emporte, car ils refusent de pratiquer le droit.
Pr 21:8- Tortueuse est la voie de l'homme criminel, mais de l'innocent l'action est droite.
Pr 21:9- Mieux vaut habiter à l'angle d'un toit que faire maison commune avec une femme querelleuse.
Pr 21:10- L'âme du méchant souhaite le mal, à ses yeux le prochain ne trouve pas grâce.
Pr 21:11- Quand on châtie le railleur, le niais s'assagit; quand on instruit le sage, il accueille le savoir.
Pr 21:12- Le Juste considère la maison du méchant : il précipite les méchants dans le malheur.
Pr 21:13- Qui ferme l'oreille à l'appel du faible criera, lui aussi, sans qu'on lui réponde.
Pr 21:14- Un don secret apaise la colère, un présent sous le manteau, la fureur violente.
Pr 21:15- C'est une joie pour le juste de pratiquer le droit, mais c'est l'épouvante pour les malfaisants.
Pr 21:16- Qui s'égare loin du chemin de la prudence dans l'assemblée des ombres reposera.
Pr 21:17- Restera indigent qui aime le plaisir, point ne s'enrichira qui aime vin et bonne chère.
Pr 21:18- Le méchant est la rançon du juste; à la place des hommes droits : le traître.
Pr 21:19- Mieux vaut habiter en un pays désert qu'avec une femme querelleuse et chagrine.
Pr 21:20- Il y a un trésor précieux et de l'huile dans la demeure du sage, mais le sot les engloutit.
Pr 21:21- Qui poursuit la justice et la miséricorde trouvera vie, justice et honneur.
Pr 21:22- Le sage escalade la ville des guerriers, il abat le rempart dans lequel elle se confiait.
Pr 21:23- A garder sa bouche et sa langue, on se garde soi-même de l'angoisse.
Pr 21:24- Insolent, hautain, son nom est " railleur "! il agit dans l'excès de son insolence.
Pr 21:25- Le désir du paresseux cause sa mort, car ses mains refusent le travail.
Pr 21:26- Tout le jour l'impie est en proie au désir, le juste donne sans jamais refuser.
Pr 21:27- Le sacrifice des méchants est une abomination, surtout s'ils l'offrent avec malice.
Pr 21:28- Le faux témoin périra, mais qui sait écouter parlera à jamais.
Pr 21:29- Le méchant se donne un air assuré, l'homme droit affermit sa propre conduite.
Pr 21:30- Il n'y a ni sagesse, ni intelligence, ni conseil devant Yahvé.
Pr 21:31- On équipe le cheval pour le jour du combat, mais c'est à Yahvé qu'appartient la victoire.
